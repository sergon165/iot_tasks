Интерполяция временного ряда

Данный сайт создан для работы интерполяции временных рядов с заданным интервалом. Загрузите CSV файл с вашим временным рядом, укажите необходимый интервал и measurement, куда будут записаны данные. Исходные данные будут записаны в (measurement)_raw, обработанные - в (measurement)_new.
